Instalação

Para instalar o Inter, não é necessário fazer mil configurações, basta importar o Inter a partir do CDN(jsDelivr) ou baixar ele e usar directamente a partir da sua máquina.

CDN

É óbvio que na parte do link onde estiver escrito numero-da-versao você deve escrever o número da versão, por exemplo se nós quisermos usar a versão 2.0.12, lá devemos escrever v2.0.12, viu? Deve começar com a letra v.

Localmente

Queres usar o Inter directamente em sua máquina? É super simples, baixa ele clicando em Baixar - v2.0.16, e agora é só importar ele em uma tag script.

Você pode ter se perguntado, se eu quiser baixar uma versão antiga, onde encontro? Você pode encontrar todas versões do Inter nas Releases do repositório do Inter no Github. Não use uma versão antiga, excepto se for realmente necessário.

Importe o Inter sempre no final da página, e a tag script com o teu código deve aparecer depois da tag script que importa o Inter. É só isso! Agora vamos pôr a mão no código?
